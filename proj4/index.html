Overview

In this project, we learned how to model fabrics as a mesh of springs that are governed by physical laws that we model using Verlet Integration. First, we implemented the various spring connections between vertices in a plane of cloth. Next, we were able to implement physics simulations using numerical Verlet Integration. Next, we were able to add lighting to our cloth by implementing various shaders with includes simple, diffuse, phong, bump, displacement, and texture mapping. Through the various shaders, we learned about the components that make up shader program including the various contributions of different light phenomenon (diffuse, ambient, specular). Lastly, it was interesting to see how physical parameters such as the spring constant and density of the cloth influences how it behaves under physics simulations. It was interesting to see also how the various diffuse/specular constants affected the visual look of the cloth in phong, bump, displacement mapping. Just by tweaking said parameters the cloth takes on different feels ranging from shiny and lustrous to dull and rubbery.

At low damping values, the cloth falls very quickly has a good amount of swaying back and forth after it has fallen. At high damping values, the cloth falls extremely slowly and stiffly stops.

Explain in your own words what is a shader program and how vertex and fragment shaders work together to create lighting and material effects.

A shader program is a type of isolated, graphics program that manipulates the vertex's position and color using vectors and matrices. The vertex shader calculates the position, normal, and other space properties and transforms vertices. The fragment shader calculates the color that a vertex should have.

Explain the Blinn-Phong shading model in your own words. Show a screenshot of your Blinn-Phong shader outputting only the ambient component, a screen shot only outputting the diffuse component, a screen shot only outputting the specular component, and one using the entire Blinn-Phong model.

The Blinn_phong shading model has a light effect with 3 parts: uniform ambient light, diffuse light, and specular light. Uniform ambient light hits every vertex. Diffuse light doesn't depend on the viewing direction and is calculated using Lambertian law. Specular light simulates the reflected light.
